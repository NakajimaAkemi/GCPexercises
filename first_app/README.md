# Creating our first GCP application
In this section we will introduce how to create an application and deploy it to the GCP.

## Setting up for the deployment
```
flask-app/ 
  ├── app.py 
  ├── app.yaml
  ├── static/
    └── hello.css 
  ├── templates/ 
    ├── hello.html 
    └── 404.html 
  ├── requirements.txt 
  └── README.md
```
